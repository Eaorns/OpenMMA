Simple, open and free software for associations

Looking for an affordable and easy to use system for your member- and event-management? OpenMMA (member management and administration) is an open source system, which means the code is freely accessible to everyone to use and adapt as desired. The project was started based on the philosophy that everyone should have access to good code, and to offer an alternative to existing commercial member management systems. In short: free software which you can adapt entirely to your needs!

  • How much does it cost?

    Nothing! You do not have to pay to use the system. All the code can be found on GitHub, together with an extensive manual. For more information, have a look at the FAQ.

  • Which features does the system have?

    A more extensive overview of all features can be found below, but in short: OpenMMA has the same features as other member management systems. Bookkeeping has not been added yet.

  • When will I be able to use it?

    We are currently still developing the system, but we are trying to have the first version ready by early April. However, the great thing about the fact that we are still in this phase of the development process is that we can also take all of your ideas and wishes into account!

Member pages

Your members get their own page where they can easily change personal information, like their phone number. They also can register themselves for activities through their account.

Calendar

Using the event calendar, members can easily see which events are going to take place. They also can register with a single click!

Groups & Roles

You can easily add your members to committees. If you want, you can give users in committees specific rights, like adding events to the calendar.

Registration form

New members can register using a simple form on your website, and automatically get put into the database. The only thing you have to do is verify their registration!

Fully customizable

Every association is different. OpenMMA allows you to customize and adjust the system exactly to your needs. Entirely tailor-made!

Free

Pay what you want, if you want. You get full access to the code and manual without any costs. This is everything you need to start using the system and set it up exactly how you need it. However, if you need help doing this, we can do it for you. Have a look at the FAQ for more information.

Open source?

You may wonder why we make the code for OpenMMA freely available. We did this based on the idea that code is not something to earn money with, but something to help people move forward. With open source code, everyone can see exactly how software works, which means there won't be any shady practices, as they would get discovered quickly. Openness and transparency are very important!

Other popular open source projects include, for example, Wordpress (which you can use to make webistes), Firefox (a webbrowser like Chrome) and Android. All these projects were created based on the idea that people should have free access to code.

Veelgestelde vragen

Open source is ook een filosofie. Wij vinden het belangrijk dat software beschikbaar is voor iedereen, en vanuit die overtuiging willen wij een goed, toegankelijk systeem aanbieden waar iedereen aan kan bijdragen. Daarnaast zien wij het ontwikkelen van OpenMMA als een mogelijkheid om meer ervaring op te doen, en wat bij te dragen aan de samenleving.

Ons doel is om rond het begin van april 2023 een bruikbaar product te hebben.

Dit hangt natuurlijk af van uw huidige systeem. De eerste versie van het systeem zal beperkte functionaliteit hebben, omdat we prioriteit geven aan het zo snel mogelijk leveren van een bruikbaar product. We zullen in de maanden daarna zo veel mogelijk andere functionaliteit gaan implementeren.

Kort samengevat wordt het met de eerste versie mogelijk om ledenaccounts te hebben (en voor nieuwe leden om zelf accounts te maken), de mogelijkheid tot het maken van groepen met rollen binnen die groepen, het plannen van events/activiteiten en een inschrijfmogelijkheid voor leden, het bijhouden van overige informatie over leden (wat voor informatie dit is kan ingesteld worden), en het importeren van accounts uit een extern systeem. Daarna zullen we overige functionaliteiten gaan implementeren, zoals het integreren van betaalmogelijkheden, custom layouts en opmaak, integratie met bijvoorbeeld Wordpress maar ook de mogelijkheid om de gehele website via OpenMMA te beheren, en nog meer! Als er verzoeken zijn voor bepaalde features willen we deze graag meenemen! Een overzicht van de geplande functionaliteiten kun je ook op github vinden.

OpenMMA zal je dezelfde functionaliteit bieden als een commercieel product, maar dan gratis! Ook hoef je je niet druk te maken over 'premium'-abonnementen. Daarnaast steun je de open source filosofie en kun je het volledig aanpassen naar je eigen wensen, danwel door mensen binnen je vereniging met technische kennis, danwel met onze hulp of een andere externe partij. Voor open source projecten is de community heel belangrijk. Hoe meer gebruikers, hoe beter het systeem zal worden door de aandacht die het krijgt en de mensen die het daarmee trekt.

Als je zelf niet de mogelijkheid hebt om de hosting te regelen en het systeem te installeren, bieden we het aan als service om dit voor je te regelen. Hiervoor rekenen we 5 euro per maand (ex. btw). We zorgen daarvoor verder ook dat het systeem up-to-date gehouden wordt.
